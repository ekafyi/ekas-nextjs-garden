Use JavaScript to detect whether user prefers dark colour scheme by default. Returns boolean.

```js
window.matchMedia("(prefers-color-scheme: dark)").matches;
// true
```

We can then assign the appropriate CSS class (eg. `<body class="is-dark">`) or do other things to enable the appropriate colour theme/mode.
