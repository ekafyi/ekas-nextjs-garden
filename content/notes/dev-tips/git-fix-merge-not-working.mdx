Both `git merge my-working-branch` and `git pull origin my-working-branch` (the working branch has been pushed) returns `Already up to date.`.

The files in both branches _are_ different, both on local and remote as seen in the web-based Github repo.

**Solution:** `git checkout my-working-branch -- .`

Changed files are staged into the current branch. You have to add and commit these files to this branch instead of having the original commits merged here.
