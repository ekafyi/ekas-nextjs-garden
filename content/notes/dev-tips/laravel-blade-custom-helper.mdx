In this example, we are making a helper to format text in our Blade template.

1. Create a file in `app/Helpers/MyCustomHelper.php`

```php
<?php
namespace App\Helpers;

class MyCustomHelper
{
	static function shout($string)
	{
		return ucwords($string);
	}
}
```

2. Add to `config/app.php` in the `aliases` array

```php
'aliases' => [
  // ...
  'MyCustomHelper' => App\Helpers\MyCustomHelper::class
],
```

3. Use it in your Blade template

```php
{!! MyCustomHelper::shout("Your text here") !!}

// YOUR TEXT HERE
```
